## Projects

### 1. Multi-Layer Neural Network for FashionMNIST Classification
- Built an MLP from scratch using NumPy
- Achieved 88% test accuracy with tanh activation, momentum, early stopping, and cross-entropy loss
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/Building%20a%20Neural%20Network%20for%20Multi-Class%20Classification/CSE_151B_251B__Deep_Learning_Winter_2025___Programming_Assignment_1%20(1).pdf)

### 2. Semantic Segmentation on PASCAL VOC-2012
- Trained U-Net and ResNet-based FCNs in PyTorch
- Reached 72% pixel accuracy and 0.1055 IoU
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/Fully%20Convolutional%20Networks%20for%20Semantic%20Segmentation%20-%20Implementing%20Deep%20Learning%20on%20PASCAL%20VOC-2012/CSE_151B_251B__Deep_Learning_Winter_2025___Programming_Assignment_2%20(1).pdf)

### 3. Character-Level LSTM for Shakespearean Text Generation
- Built and tuned LSTM/RNN models on the TinyShakespeare dataset
- Generated coherent text with 1.378 test loss and optimal temperature scaling
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/Character%20Level%20LSTM%20for%20Shakespearean%20Text%20Generation%20with%20Neural%20Sequence%20Modeling/Character_Level_LSTM_for_Shakespearean_Text_Generation_with_Neural_Sequence_Modeling%20(4).pdf)

### 4. Optimizing BERT for Scenario Classification
- Fine-tuned BERT on the Amazon MASSIVE dataset
- Applied SupCon and SimCLR contrastive learning; achieved 91.5% accuracy
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/Optimizing%20BERT%20Representations%20for%20Scenario%20Classification%20-%20Fine-Tuning%20and%20Contrastive%20Learning%20Approaches/Optimizing_BERT_Representations_for_Scenario_Classification__Fine_Tuning_and_Contrastive_Learning_Approaches%20(6).pdf)

### 5. Deep Q-Learning for Competitive Tank Battles
- Built a custom PyGame environment with Dueling DQN and curriculum learning
- Trained agents to a 6/10 win rate against random bots
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/Optimizing%20AI%20Decision-Making%20in%20Tank%20Battles%20Using%20Deep%20Q-Learning/Optimizing_AI_Decision_Making_in_Tank_Battles_Using_Deep_Q_Learning%20(6).pdf)
