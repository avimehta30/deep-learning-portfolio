## Projects

### 1. Multi-Layer Neural Network for FashionMNIST Classification
- Built an MLP from scratch using NumPy
- Achieved 88% test accuracy with tanh activation, momentum, early stopping, and cross-entropy loss
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/FashionMNIST-MLP)

### 2. Semantic Segmentation on PASCAL VOC-2012
- Trained U-Net and ResNet-based FCNs in PyTorch
- Reached 72% pixel accuracy and 0.1055 IoU
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/PascalVOC-Segmentation)

### 3. Character-Level LSTM for Shakespearean Text Generation
- Built and tuned LSTM/RNN models on the TinyShakespeare dataset
- Generated coherent text with 1.378 test loss and optimal temperature scaling
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/Shakespeare-LSTM-TextGen)

### 4. Optimizing BERT for Scenario Classification
- Fine-tuned BERT on the Amazon MASSIVE dataset
- Applied SupCon and SimCLR contrastive learning; achieved 91.5% accuracy
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/BERT-Scenario-Classification)

### 5. Deep Q-Learning for Competitive Tank Battles
- Built a custom PyGame environment with Dueling DQN and curriculum learning
- Trained agents to a 6/10 win rate against random bots
- [📄 View Report](https://github.com/avimehta30/deep-learning-portfolio/blob/main/DQN-Tank-Battle)
